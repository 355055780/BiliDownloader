## 介绍

这是一款开源的 b 站 视频下载器

## 使用

发行版在Windows中安装即可使用，双击启动后会从作者的服务器获取更新信息和许可证，所以启动时间可能较慢

输入BV号、AV号或MD号即可获取视频 (MD号是指番剧详情页面链接上的**mdxxxx**)

在分p选取页面，输入你要下载的区段：

> 假设你要下载第一P和第二P，你可以输入：1-1, 2-2 或者输入1-2
>
> 如果要下载第一P到第五P，但是不要第三P，你可以输入：1-2, 4-5

确认后选择下载视频质量

有以下清晰度:

|输入值|清晰度|
|:-:|:-:|
|120|4K|
|116|1080p60|
|112|1080p+|
|80|1080p|
|72|720p60|
|64|720p|
|32|480p|
|16|320p|

随后可开始下载
